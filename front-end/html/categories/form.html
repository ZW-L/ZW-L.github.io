get
post		 HTML5

formnovalidate		 HTML5

_blank_

# label

  • 为标签元素定义标记，当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上
  • 不会向用户呈现任何特殊效果
  • 标签的 for 属性应当与相关元素的 id 属性相同
属性名 描述 版本
form 规定 label 字段所属的一个或多个表单。取值：
form_id		 HTML5
for 规定 label 与哪个表单元素绑定。取值：
element_id
标准属性

# textarea

  • 定义一个多行的文本输入控件，文本区域中可容纳无限数量的文本，其中的文本的默认字体是等宽字体（通常是 Courier）
  • 可以通过 cols 和 rows 属性来规定控件的尺寸大小，更好的办法是使用 CSS 的 height 和 width 属性

autofocus		 HTML5

form_id		 HTML5
maxlength 规定文本区域允许的最大字符数。取值：
number		 HTML5
placeholder 规定一个简短的提示，描述文本区域期望的输入值。取值：
text		 HTML5
required 规定文本区域是必填的。取值：
required		 HTML5
wrap 规定当提交表单时，文本区域中的文本应该怎样换行。取值：
hard
soft		 HTML5
name 规定文本区域的名称。取值：
text
disabled 规定禁用文本区域。取值：
disabled
readonly 规定文本区域为只读。取值：
readonly
cols 规定文本区域内可见的宽度。取值：
number
rows 规定文本区域内可见的行数。取值：
number
标准属性 支持 HTML 的全局属性。
全局属性 支持 HTML 的全局属性。

# option

  • 定义下拉列表中的一个选项，标签中的内容作为 <select> 或者 <datalist> 一个元素使用，否则这个标签是没有意义的
  • 可以在不带有任何属性的情况下使用，通常需要使用 value 属性，此属性会被送往服务器
  • 如果列表选项很多，可以使用 <optgroup> 标签对相关选项进行组合
属性名 描述 版本
disabled 规定此选项应在首次加载时被禁用。取值：
disabled
selected 规定选项（在首次显示在列表中时）表现为选中状态。取值：
selected
label 定义当使用 <optgroup> 时所使用的标注。取值：
text
value 定义送往服务器的选项值。取值：
text
标准属性
全局属性 支持 HTML 的全局属性。
事件属性 支持 HTML 的事件属性。
