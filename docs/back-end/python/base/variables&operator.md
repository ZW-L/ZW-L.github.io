

## 变量赋值
+ 变量不需要声明，变量赋值操作是变量声明和定义的过程
```python
a = 12
print(a)
str = 'hello world'
b = c = 10
x, y, z = 1, 2, 3
```

## 运算符
+ 算术运算符: 
  + `+`
  + `-`
  + `*`
  + `/`
  + `%`： 取模
  + `**`： 幂
  + `//`： 取整除
+ 比较（关系）运算符:
  + `==`
  + `!=`
  + `<>`
  + `>`  
  + `<`  
  + `>=`
  + `<=`
+ 赋值运算符:
  + `=`
  + `+=`
  + `-=`
  + `*=`
  + `/=`
  + `%=`
  + `**=`
  + `//=`
+ 逻辑运算符:
  + `x and y`： 逻辑与
  + `x or y`： 逻辑或
  + `not x`： 逻辑非
+ 位运算符:
  + `&`： 按位与，均为1则为1，否则为0
  + `|`： 按位或，有一个为1则为1，否则为0
  + `^`： 按位异或，不同时为1，否则为0
  + `~`： 取反，将1和0交换
  + `<<`： 左移
  + `>>`： 右移
+ 成员运算符:
  + `in`： 在指定的序列中
  + `not in`： 不再指定的序列中

+ 身份运算符:
  + `is`： 判断两个标识符是否引用自同一个对象
  + `is not`： 判断两个标识符是否引用自不同的对象

+ 运算符优先级（从高到低）:
  + `**`
  + `~ + -`
  + `* / % //`
  + `+ -`
  + `>> << (右移左移)`
  + `&`
  + `^ |`
  + `<= <> >=`
  + `<> == !=`
  + `= %= /= //= -= += **=`
  + `is is not`
  + `in not in`
  + `not or and`
