## 简介

+ python 是一门**动态**的、**面向对象**的高级程序语言
+ [官方下载](https://www.python.org/downloads/)并安装


### 编辑器

+ PyCharm
+ VS Code


### 特殊的语法

+ 代码风格
  + 使用回车代替分号（`;`）来表示表达式结束
  + 使用嵌套代替花括号（`{}`）来表示语句块
  + 单行注释(`#`)，多行注释(`'''`, `"""`)
  + 没有用于变量声明的关键字
+ 界限
  + 条件判断的括号可有可无
