## 与其他编程语言的异同

+ 代码风格
  + 使用换行符代替**分号**作为行结束符号
  + 使用缩进代替**花括号**
  + 的单行注释(#注释)，多行注释('''注释'''或"""注释""")
  + 没有用于变量声明的关键字

+ 界限
  + 条件判断的括号可有可无
