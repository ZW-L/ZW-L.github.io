## 概念

+ 定义：树（Tree）是 $n(n\geq1)$ 个节点的有限集。




## 分类

+ 二叉树
+ 动态查找树
  + 二叉查找树（Binary-Search Tree）
  + 平衡二叉树（AVL Tree）
  + 红黑树（Red-Black Tree）
  + 哈夫曼树（Huffman Tree）
+ 字典树
+ 线段树
+ 树状数组



## 遍历

+ 深度优先搜索（Depth-First-Search，DFS）
  + 前序遍历
  + 中序遍历
  + 后序遍历
+ 广度优先搜索（Breadth-First-Search，BFS）
