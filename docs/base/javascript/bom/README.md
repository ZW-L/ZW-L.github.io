## 简介

### Web API

+ [Web API](https://developer.mozilla.org/zh-CN/docs/Web/API) 主要用于 `JavaScript`，但也可能有例外。

### window

+ `window` 对象是浏览器网页的根对象，一切 `Web API` 支持的接口(对象)都继承自 `window`。


### 基本接口

+ canvas
+ console
+ document
+ frames
+ history
+ location
+ navigator
+ screen


### 热门接口

+ Storage
+ Service Workers
+ WebGL
