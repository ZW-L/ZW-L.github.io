## 介绍

[HTMLElement](https://developer.mozilla.org/zh-CN/docs/Web/API/HTMLElement)：

+ 继承并扩展了 `Element` 接口，表示所有的 `HTML` 元素
+ 所有 `HTML` 元素都直接或简介实现了 `HTMLElement` 接口
