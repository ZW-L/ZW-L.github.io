## 取值

属性|说明
-|-|-
`<color>`|颜色名, hex, rgb(), rgba(), hsl(), hsla()|
`<string>`|字符串，可包含转义字符串|
`<url>`|url("example.css"), url(example.css)|
`<identifier>`|区分大小写|
`<number>`|浮点数|
`<integer>`|整数|
`counter()`|用于 content 属性的函数|
`attr()`|用于插入元素值的函数|
`<length>`|长度单位|
`<percentage>`|百分比单位|
`<angle>`|角度单位，取值：deg, grad, rad, turn|>IE8
`<time>`|时间单位，取值：s, ms|>IE8
